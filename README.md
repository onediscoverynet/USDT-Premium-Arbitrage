# USDT Premium Arbitrage

안전하고 투명한 확정 수익형 투자 모델.
City of Dreams 호텔에서 오프라인으로 진행되는 안전한 암호화폐 차익거래 제안입니다.

## 프로젝트 소개

이 프로젝트는 USDT(테더)와 PHP(페소) 간의 시세 차익을 이용한 오프라인 차익거래(Arbitrage) 시스템을 소개하는 랜딩 페이지입니다.

## 주요 기능

- **반응형 웹 디자인**: PC 및 모바일 최적화
- **수익 계산기**: 투자 금액 및 환율에 따른 예상 수익 실시간 계산
- **고급스러운 UI**: 골드 & 블랙 테마의 프리미엄 디자인
- **애니메이션 효과**: 스크롤 및 등장 애니메이션 적용

## 기술 스택

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Vanilla JS)
- Font Awesome (Icons)
- Google Fonts (Noto Sans KR)

## 설치 및 실행

1. 이 저장소를 클론합니다.
   ```bash
   git clone https://github.com/onediscoverynet/USDT-Premium-Arbitrage.git
   ```
2. `index.html` 파일을 브라우저에서 실행합니다.
